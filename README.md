# Cat's Taste 訂單 App 同事使用手冊

呢個 app 係香港貓咪博覽會展場用嘅送貨訂單記錄工具。無網絡都可以落單、複製 WhatsApp 訂單確認文字、記錄快遞單號；有網絡時會同步到 Google Sheets。

## 手機安裝

### iPhone / iPad
1. 用 Safari 開啟 app 網址。
2. 按分享按鈕。
3. 選擇「加入主畫面」。
4. 之後直接喺主畫面開 `CT訂單`。

### Android
1. 用 Chrome 開啟 app 網址。
2. 按右上角選單。
3. 選擇「新增至主畫面」或「安裝應用程式」。
4. 之後直接喺主畫面開 `CT訂單`。

## 落單流程

1. 喺「落單」頁選產品。
2. 產品數量分兩邊：
   - 左邊：原箱
   - 右邊：單件
3. 長按 `+` 可以快速加數量。
4. 底部會自動計算小計、折後金額、贈品及免費送貨。
5. 按「落單」。
6. 彈窗上方會顯示收據摘要，確認產品、數量及價錢無錯。
7. 填客人姓名、電話、地址。
8. 按「確認落單」。
9. 去「訂單」頁按「確認WA」，複製文字後貼去 WhatsApp 俾客人確認。

## 訂單及執貨

- 「訂單」頁會顯示所有本機訂單及已拉取嘅共享訂單。
- 訂單狀態可改為：
  - 待確認
  - 已確認
  - 已發貨
- 可以標記或取消：
  - 已發 WA 確認
  - 已發 WA 快遞
- 如要執貨，按「詳情」睇完整產品清單。
- 訂單列表及詳情入面嘅電話號碼可以直接按，會用 WhatsApp 開啟；未輸入國碼時會預設香港 `+852`。

## 快遞單號

1. 喺訂單卡按「快遞」。
2. 可以手動輸入單號，或者按相機按鈕掃描 barcode / QR code。
3. 如果 QR code 內容係快遞連結或文字，app 會盡量抽出當中嘅快遞單號。
4. App 會自動識別：
   - SF / 順豐
   - JD / 京東
   - 其他快遞
5. 確認後，系統會生成 WhatsApp 快遞通知文字。
6. 複製後貼去 WhatsApp 通知客人。

## Google Sheets 同步

### 第一次設定
1. 開 Google Sheet。
2. 到「擴充功能」>「Apps Script」。
3. 將 `Code.gs` 內容貼入 Apps Script。
4. 儲存後部署為 Web App。
5. 如之後更新過 `Code.gs`，要到「部署」>「管理部署」>「編輯」>「版本」選「新版本」再部署；否則手機仍會跑舊 Apps Script，可能會拉取失敗。
6. 權限建議：
   - Execute as：Me
   - Who has access：Anyone with the link
7. 複製 Web App URL。
8. 回到 app「後台」>「Google Sheets」>「Web App」，貼上 URL。
9. 按「測試連接」，見到成功就可以用。

### 日常同步
- 有網絡時，落單、改狀態、標記 WA、填快遞會自動嘗試同步。
- 如果場內網絡差，可先照常落單；見到網絡恢復後，按訂單頁右上角雲端同步按鈕。
- 如另一部手機有新訂單或更新，按訂單頁右上角拉取按鈕。
- 如果手機主畫面 app 一直用舊設定，先關閉再重開；仍然唔得就刪除主畫面 icon，再用 Safari/Chrome 重新加入主畫面。

## 注意事項

- 免費送貨以折後滿 `$800` 計。
- 原箱價不可與九折優惠同時享用。
- App 無付款狀態，收銀付款請跟現場原本流程處理。
- 「刪除訂單」會標記為「已刪除」，訂單仍會保留喺紀錄入面；同步後其他同事都會見到「已刪除」狀態。
- 展場開始前，建議每部同事手機先測試一次落單、同步、拉取及快遞單號。
